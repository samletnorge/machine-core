# Machine Core

Machine Core is a Python package for building AI agents with large language models and embeddings. The library provides abstractions to simplify creating and deploying agents such as chat, CLI, receipt processing, tweets, etc.

## Documentation

- **Installation**: `pip install machine-core`
- **Configuration**: edit `mcp.json` for model settings.
- **Usage**: Import agents from `machine_core.agents`.
- **Example**: see `src/machine_core/agents/examples.py`.

For more detailed docs, refer to the source code and the README.
